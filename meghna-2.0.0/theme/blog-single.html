10年來最重大的畫面更新，《Minecraft》支援光線追蹤

  • 成為神
  • July 18, 2022

自從宣佈這項顛覆以往的更新以來，Nvidia在網路上看到玩家不斷反覆提出某些問題，Nvidia曾加以彙整來解答。


問: Java 版《Minecraft》的光影包與 Windows 10 (Bedrock) 版的光線追蹤有什麼差異？

答: 光影包是透過近似、模擬和 pre-baked 效果來增強《Minecraft》的繪圖技術，並以升級的渲染器、新紋理和其他增強功能進行整合， 然而這些功能最終會受到《Minecraft》的模組限制，而且無法存取《Minecraft》的原始碼和核心檔案。

Nvidia與 Mojang 和 Microsoft 合作，將光線追蹤視為核心遊戲功能進行開發，並直接加入引擎，使其能以更快的畫面播放速率運用完整的 GPU 加速 (包含 RT 核心) 功能，而畫質細膩的即時光線追蹤效果也能改善遊戲各方面的繪圖。



問: 與光影包相比，光線追蹤有哪些優點？ 答: 如前所述，光影包是由第三方加入的增強功能，因此具有先天限制。例如，許多效果是在畫面空間中執行，這表示效果只能影響或受到當前場景中可見的元素影響。

相對的，光線追蹤可以讀取和考量遊戲繪圖的所有元素，無論元素位在玩家的鏡頭上方、下方還是後方都不會遺漏，因此能夠呈現逼真的反應和互動，以及符合真實物理情況的光線、陰影和效果。透過直接整合至引擎，而不是作為模組新增在最上層，可以結合新技術、效果和功能來提供更好的遊戲體驗。

然而，光影包和材質模組包仍然很重要，因為這些功能讓創作者可以為遊戲打造和指定非常特定的「外觀」，而預設的光線追蹤整合將會透過最尖端的效果和功能實現 Mojang 理想中的《Minecraft》。因此，我們期待模組玩家能與 Bedrock 合作，針對光線追蹤效果和材質進行修改，並以驚人的逼真度將想像化為現實。

例如，想想看卡通渲染結合光線追蹤和高解析度物理材質會有什麼效果，或者使用可以透過光線追蹤效果徹底改變遊戲氛圍的風格化模組。

Microsoft 的業界標準 DXR 光線追蹤 API 需要 DirectX 12，而 Bedrock 已有支援 DirectX 12。

問: 要如何在《Minecraft》中啟用光線追蹤？

答: 遊戲內會提供選項。你的系統上需要有最新版本的 Windows 10、支援硬體即時光線追蹤的 GPU、以及《Minecraft》基岩版。

然而光線追蹤顯示卡推出時高昂的售價讓這項技術難以得到普及。
好不容易到了更新換代的時刻，正當光線追蹤顯示卡正要飛入尋常百姓家時，突如其來的加密貨幣礦潮毀掉了一切。
遭受有錢玩家、顯卡廠商、礦工、黃牛的冷嘲熱諷，忍辱負重的普通玩家在2年後終於迎來了加密貨幣礦難。

經過了玩家歷史上最困難的2020~2022，也許是時候在加密貨幣礦難後為自己買一張足夠便宜（*2）的光線追蹤顯示卡了。





*2：你不買，我不買，明天還能降1000。
